### 1. 项目名称 (Project Name)
- 魔改 ResNet18：集成 SE-Block 与残差连接改进的图像分类研究
### 2. 项目角色 (Project Role)
- 独立开发者 / 核心算法实现
### 3. 项目描述 (Project Description)
本项目旨在通过修改经典 ResNet18 网络拓扑结构，提升模型在复杂图像任务中的特征提取能力。

- 核心架构 ：基于 PyTorch 框架，在残差块中嵌入 Squeeze-and-Excitation (SE) 通道注意力模块 ，通过全局信息建模实现特征通道的自动加权。
- 工程架构 ：采用模块化设计（Model/Dataset/Config 分离），实现配置文件驱动的实验管理，并集成 TensorBoard 实时监控训练动态。
- 环境支持 ：利用 Apple Silicon (MPS) 进行硬件加速，优化大规模实验的迭代速度。
### 4. 项目业绩 (Project Achievements) —— 这是面试官最看重的“干货”
- 模型性能魔改 ：通过引入 SE 注意力机制，在仅增加约 0.8% 参数量 （11.17M → 11.26M）的前提下，成功将 CIFAR-10 验证集准确率从基础版的 [XX]% 提升至 91.06% 以上。
- 工程化治理 ：构建了 生产级训练流水线 ，实现了模型 Checkpoint 自动保存、数据加载异常捕获（处理损坏样本）及显存溢出 (OOM) 的优雅降级处理。
- 过拟合深度优化 ：针对极深网络在小数据集上的过拟合问题，综合运用 Label Smoothing (0.1) 、 Weight Decay (5e-4) 及 Cosine Annealing (余弦退火) 学习率调度算法，有效收窄了 Train-Val Gap。
- 黑盒可视化 ：开发了 注意力权重追踪工具 ，通过可视化分析验证了模型在不同深度下对特征通道的选择性关注逻辑，增强了算法的可解释性。
